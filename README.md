## 1. Описание проекта

Статический персональный сайт-визитка, созданный с помощью HTML5 и CSS3.
Сайт содержит три страницы: главную, «Обо мне» и «Контакты». Проект
открывается локально в браузере без установки зависимостей и без сервера.

## 2. Цель практической работы

Практическая работа направлена на освоение Git и GitHub:

- инициализация репозитория;
- создание коммитов;
- работа с ветками;
- публикация изменений (push);
- создание pull request;
- слияние веток (merge).

Сайт служит учебным материалом для демонстрации этих процессов.

## 3. Возможности сайта

- адаптивная вёрстка для компьютера и мобильных устройств;
- единый стиль на всех страницах;
- доступная навигация с отметкой текущей страницы;
- секции с навыками и описанием проекта;
- страница «Обо мне» с целями обучения и принципами работы;
- страница контактов со ссылкой на GitHub и демонстрационной формой
  (без отправки данных).

## 4. Структура проекта

```text
git-practice/
├── index.html      # Главная страница
├── about.html      # Страница «Обо мне»
├── contact.html    # Страница «Контакты»
├── style.css       # Общие стили
├── README.md       # Описание проекта
└── .gitignore      # Игнорируемые файлы
```

## 5. Использованные технологии

- HTML5
- CSS3 (Flexbox, Grid, CSS-переменные)
- Git
- GitHub

В проекте нет JavaScript, фреймворков, сборщиков и внешних пакетов.

## 6. Запуск проекта

Откройте файл `index.html` в любом современном браузере
(двойной щелчок по файлу или через меню «Открыть файл»).

Установка пакетов не требуется — у проекта нет зависимостей.

## 7. Git-ветки

| Ветка | Назначение |
| --- | --- |
| `main` | Стабильная ветка с готовой версией сайта |
| `feature/page` | Ветка для разработки дополнительных страниц |

Проект демонстрирует типичный учебный сценарий: инициализация репозитория,
коммиты, создание веток, push, pull request и слияние изменений.
